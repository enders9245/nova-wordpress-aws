# Instalación y configuración de WordPress

## Stack utilizado
- Amazon Linux 2023
- Apache (httpd)
- PHP 8.x
- WordPress (última versión, instalado vía user data script en EC2)

## Pasos realizados
1. Lanzamiento de instancias EC2 con user data que instala httpd, PHP y WordPress.
2. Conexión a la base de datos RDS (MySQL).
3. Finalización del instalador de WordPress vía navegador.
4. Creación de usuario administrador del sitio.
